## 这是什么？

一个装在 [Hermes Agent](https://hermes-agent.nousresearch.com/docs) 上的 skill。你只需要说一句：

> 「帮我看看有没有新商机」

AI 会自动按下面这套体系，扫描 7 大信号源、跑 14 条中英文 web_search、给你产出 **5 个打分后的候选商机 + 触发动作 + 反模式警告 + 黑天鹅预案**。

底层判断标准来自 **「00后富一代」知乎语录（566 篇 / 9.3 万字）**：
- 轻资产 + 1% 预算试错
- 信息差 = 钱
- 跨市场套利 5x~10x 收益
- 反消费主义 + 延迟满足
- 名 = 放大器

## 内置的判断体系

### 5 条金线
1. **轻资产试错** — 单笔 ≤ 总资产 1%
2. **信息差 = 钱** — 找 100% 散户会接盘的时间窗口
3. **跨市场套利** — 同样的能力搬到竞争更弱的市场 = 5x~10x
4. **反消费主义** — 花钱不超过收入 10%
5. **名 = 放大器** — 个人 IP 化

### 7 大信号源
- **S1** 跨语言/跨国价差（中文 vs 英文 / 国内 vs 国外）
- **S2** 跨平台/跨地区定价差
- **S3** 跨时代/世代刚需（老年人 / Z 世代）
- **S4** 跨合规套利（A 阵营法律管不了 B 阵营的生意）
- **S5** 跨周期套利（萧条/复苏/过热/滞胀切换）
- **S6** 跨技术栈套利（程序员能力外溢）
- **S7** AI 时代新刚需

### 6 维商机打分卡（每项 0-10 分，总分 ≥ 42 才推荐）
| 维度 | 含义 |
|---|---|
| D1 试错成本 | ≤ 1% 总资产 = 10 分；需贷款梭哈 = 0 分 |
| D2 经验可回收 | 失败也涨经验 = 10 分 |
| D3 无货源/中间商 | 撮合/倒买 = 10 分；重资产 = 0 分 |
| D4 英文+GFW 加成 | 完全靠跨境 = 10 分 |
| D5 新兴市场 | AI/加密/跨境 = 10 分；房地产 = 0 分 |
| D6 可 IP 化 | 能做个人 IP + 全球化 = 10 分 |

### 13 条反模式（硬性一票否决）
1. 梭哈 / 贷款梭哈
2. 重资产创业（开店开厂）
3. 甩手掌柜
4. "与世无争慢慢变富" 的心态
5. 听比自己 low 的人的财务建议
6. 长持一种叙事 / 一个币
7. 看到商机就找理由证明"做不成"
8. 投公募基金 / 长期定投
9. "中式低调隐忍"
10. 跟中文区 KOL 学交易
11. 在公开场合炫耀赚的钱来自谁
12. 不挂个人 IP
13. 不脱离英文 + GFW + Twitter 三角
