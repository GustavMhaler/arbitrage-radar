# 商机雷达 v1 (Arbitrage Radar)

> "最好的赚钱方式就是不要去陷入内卷，去找到那个其他人想不到的地方，在那里散发光芒。一般来说这种机会要么是跨行业，要么就是在新兴市场。"  —— 00后富一代 #24

## 触发条件

用户出现以下意图时立刻触发：

- 「帮我看看 / 扫一下 / 找找有什么新商机 / 赚钱点子 / alpha / 套利机会」
- 「我想搞个副业/小生意，从哪里开始？」
- 「如何用信息差赚钱」
- 「轻资产低成本的生意有什么」
- 「xx 行业还有机会吗」
- 「跨市场/跨境/出海能做什么」
- 「现在 / 2025 / 2026 还能做什么赚钱」

**不要**在用户问「这个股票能不能买」「帮我写代码」「翻译一下」时误触发。本 skill 只做**机会发现**，不做交易信号、投资建议或代码生成。

## 核心判断体系 (5 条金线)

来自语录库提炼，缺一不可：

| # | 原则 | 含义 | 语录出处 |
|---|---|---|---|
| 1 | **轻资产试错** | 单笔投入 ≤ 总资产 1%，输了不痛不痒还能涨经验 | #1, #117, #180 |
| 2 | **信息差 = 钱** | 找到 100% 散户/接盘侠会买的时间窗口 | #1, #17, #224 |
| 3 | **跨市场套利** | 同样的能力/商品搬到竞争更弱的市场 = 5x~10x 收益 | #24, #118 |
| 4 | **反消费主义** | 花钱不超过收入 10%，把赚的钱滚回去买生息资产 | #124, #105, #95 |
| 5 | **名 = 放大器** | 个人 IP 化，让名引来更多利 | #93, #138, #3 |

## 7 大信号源 (从语录里抽出来的商机捕捉方向)

每次扫描时，**至少覆盖 5 个信号源**才能给出有价值的报告：

| # | 信号源 | 抓什么 | 经典案例 (语录) |
|---|---|---|---|
| S1 | **跨语言/跨国价差** | 英文世界有而中文世界没有，或反过来；同一商品两国价差 | 海外公司注册 #1、海外银行开户 #147 |
| S2 | **跨平台/跨地区定价差** | 同一产品在不同平台/地区的价格差 | 知乎中美区推广套餐差 900 vs 710 元 #126 |
| S3 | **跨时代/世代刚需** | 老一辈有钱人 / 年轻 Z 世代的特定刚需 | 跨国资产配置顾问 #4、AI 复活亲人 #4 |
| S4 | **跨合规套利** | A 阵营法律管不了 B 阵营的生意 | 盗版美剧在正义联盟无人管 #118、GPU 倒卖 #118 |
| S5 | **跨周期套利** | 萧条/复苏/过热/滞胀 4 阶段切换时的资产切换 | 美债+大宗商品+优质股权轮动 #2, #38 |
| S6 | **跨技术栈套利** | 把 A 行业技术用到 B 行业 | 程序员爬虫去抢鞋 #24、Midjourney 出 logo #160 |

## 商机打分卡 (6 维 0-10 分)

每个候选商机都要被打分，**总分 ≥ 42 (满分 60) 才推荐**：

| 维度 | 0 分 | 10 分 |
|---|---|---|
| D1 试错成本 | 需要贷款梭哈 | ≤ 1% 总资产，输了不痛 |
| D2 经验可回收 | 失败 = 钱打水漂 | 失败也涨经验/人脉/技能 |
| D3 是不是无货源/中间商 | 重资产开店 | 撮合/倒买倒卖/纯信息中介 |
| D4 英文+cross GFW 加成 | 完全国内封闭 | 完全靠跨境信息差 |
| D5 是否在新兴市场 | 房地产/传统餐饮 | AI/加密/跨境/新能源 |
| D6 可 IP 化 + 全球化 | 一次性卖货 | 能做个人 IP + 资产全球化 |

## 操作流程 (用户输入 → 输出)

### Step 1：识别用户的"约束条件"
向用户（或推断）明确：
- **本金量级**（< 10万 / 10-100万 / 100万+）
- **技能背景**（程序员 / 设计师 / 销售 / 学生 / 其他）
- **风险偏好**（必须 0 风险 / 可承受波动 / 能扛 -50%）
- **时间预算**（每天 2h 副业 / 全职 / 周末党）
- **地理限制**（人在国内 / 肉身已翻 / 计划出海）

如果用户没说全，给一个**默认画像**（程序员，< 10 万本金，副业形式，0 风险承受力，国内），并在报告里明确写出"假设画像是 XX，如果你不是请告诉我"。

### Step 2：用 web_search 扫描 5 个信号源
对至少 5 个信号源（从 S1~S7 里选），每个信号源跑 1~2 条 web_search query：
- "latest arbitrage opportunity 2026"
- "cross-border small business ideas 2026"
- "AI micro-SaaS opportunities"
- "跨境电商小生意 2026"
- "crypto arbitrage opportunity 2026"
- "海外华人服务 信息差"
- "AI 应用 副业"
- "新兴行业 套利 2026"

**关键：** 至少 2 条用英文搜，因为他强调要 cross the great firewall 看英文世界的前沿信息。

### Step 3：生成 5 个候选商机
每个候选必须包含：
1. **名字**（一句话描述）
2. **属于哪个信号源**（S1~S7）
3. **本金门槛 / 试错成本**（具体数字）
4. **预期年化 / 月化收益区间**（基于语录里类似案例的实测）
5. **6 维打分**（D1~D6 各 0-10 分 + 总分）
6. **触发动作**（具体第一步，比如"先注册 X 账号看竞品"）
7. **反模式检查**（13 条全部勾一遍，写出 PASS / 哪条 FAIL）
8. **风险与黑天鹅**（参考语录里的踩坑案例，比如 Bybit 暴雷 #12）
9. **可 IP 化路径**（怎么变成个人 IP）
10. **语气仿写段落**（最后用他的语气写 1-2 句"金句"，让人上头）

## 边界与失败模式

1. **不要给具体股票/币种买卖建议** —— skill 只做机会发现，不做交易信号
2. **不要推荐灰色/明确违法项目**（毒品/诈骗/盗版本身）—— 但**灰色边缘**可以提（如"找老外代办开海外公司"），标注"有合规风险，需自评"
3. **如果 web_search 全部失败** —— 直接用语录库里的经典案例做骨架，标"信号源离线，本次为理论框架报告"
4. **如果用户预算 ≥ 1000 万** —— 切换到语录里的"机构版"框架（#25 的 4000 万配置方案、#94 的 1000 万刀美股战略），不要给副业建议
