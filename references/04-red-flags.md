# 04 — 7 条错误模式(命中直接 PASS)

> 候选商机只要命中任何一条,**直接 PASS**,不用给第二眼。
> 这些不是 V2 的"反例清单"——它们是**红线**,灰产擦边也会触发。

---

## 1. 需要贷款梭哈 / 重资产开店

**信号**:
- "投资 30 万开个奶茶店"
- "贷款 50 万盘个店面"
- "加盟 XX 品牌,启动资金 100 万"

**为什么 PASS**: 违反 P1 轻资产,输了直接报废。

**怎么改**: 改成"先用 1 万块在家试奶茶配方,小红书发测评,看有没有复购"。

---

## 2. 来源不可验证的"内部消息"

**信号**:
- "群里大佬说某股票要拉"
- "某币马上要拉盘"
- "认识一个人能搞到内部渠道"

**为什么 PASS**: 信息源不可证伪 = 骗局 / 操纵市场 / 自己是韭菜。

**怎么改**: 用 web_search 公开验证 / 看 SEC 文件 / 看上市公司公告。

---

## 3. 一次只能赚一次的零和游戏

**信号**:
- 单纯倒卖某件商品(不形成 IP)
- 单纯刷量刷单(平台会封)
- 单纯"加群卖资料"(复购为 0)

**为什么 PASS**: 没有 D6(IP 化),做完一次就拉倒,不算"机会"算"零工"。

**怎么改**: 加一个"我同时提供 X 服务"的复购项,让客户下次还找你。

---

## 4. 需要先付"会员费/培训费/加盟费"

**信号**:
- "交 1999 入会送教程"
- "加盟我们送流量"
- "买课教你月入 10 万"

**为什么 PASS**: 这是 P4 反消费主义的经典陷阱——你成了别人的现金流,而不是机会发现者。

**怎么改**: 凡是要求"先交钱再教你赚钱"的,**默认视为骗局**。真想学,找 B 站免费教程。

---

## 5. 灰色 / 明确违法

**信号**:
- 毒品 / 色情 / 赌博 / 盗版本身
- 翻墙工具销售
- 假证 / 假流水 / 假评论
- 跨境洗钱 / 代收代付
- 盗版影视 / 盗版书 / 盗版软件销售
- 撸毛 / 空投 / 链上脚本(法律灰色)

**为什么 PASS**: 不属于"边缘机会标注合规"的范畴,**直接拒**,不展开讨论。

**例外**: 跨境电商/海外注册/海外银行开户这类,如果操作合规 + 用途合法,可以提,但必须明确标注"需自评合规"。

---

## 6. 不在新兴市场 / 已经是红海

**信号**:
- "现在做淘宝还来得及吗"(2015 年的问题)
- "开家线下书店"
- "做个公众号写鸡汤"
- "教人炒股(2026 年)"

**为什么 PASS**: 红海 = D5 新兴市场分数 ≤ 3,即使 D1/D2 高,总分也上不去。

**怎么改**: 选 5 年内增长 > 20%/年的赛道(AI / 跨境 / 独立开发者 / 银发 / 新能源)。

---

## 7. 没有信息差,只有体力差

**信号**:
- "送外卖多跑几单"
- "去批发市场搬货"
- "做微商刷屏"

**为什么 PASS**: 你的"优势"只是比别人更努力,不是"知道别人不知道"。没有 D4,没有壁垒,随时被更便宜的人卷掉。

**怎么改**: 在体力基础上加"信息差维度"——比如同样是送外卖,只送高端小区 + 收集周边商家需求,转介给同事抽佣。

---

## 通用诊断流程

遇到任何候选商机,按顺序问 7 个问题:

1. 是不是要贷款梭哈? → 是 → PASS
2. 信息源可不可验证? → 不可 → PASS
3. 能不能重复赚钱? → 不能 → PASS
4. 要不要先交钱? → 要 → PASS
5. 是不是违法? → 是 → PASS
6. 在不在新兴市场? → 不在 → PASS
7. 有没有信息差? → 没有 → PASS

**7 个问题全过才进入 6 维打分卡**。
