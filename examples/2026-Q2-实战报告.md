## 5 个候选商机

### 【候选 1】★★★★★ 54/60 — 中文程序员做英文 Niche Micro-SaaS
- **信号源**：S6 跨技术栈 + S7 AI 新刚需
- **本金**：≈ 800 元（域名 + ChatGPT 订阅 + Stripe）
- **预期**：$500-$3000/月单产品，爆款 $10k+/月
- **打法**：去 Reddit r/indiehackers 找痛点 → Cursor 3 天出 MVP → Stripe 收款 → Twitter #buildinpublic
- **反模式**：全 PASS
- **金句**："中国人做英文 SaaS 就跟当年白求恩去中国一样，同样的本事搬到英文世界直接降维打击。"

### 【候选 2】★★★★★ 54/60 — 海外公司注册+银行开户中介
- **信号源**：S1 跨语言价差 + S3 跨世代刚需
- **本金**：0 元
- **预期**：3k-1万/月（按 1 人头 100-300 美元算）
- **打法**：Fiverr 找 300 元上家 → 闲鱼/小红书挂 1500-3000 元 → 知乎 SEO 截流
- **反模式**：全 PASS
- **金句**："海外公司注册我卖的不是服务，是信息差。英文好的中国人就是会下金蛋的鹅。"

### 【候选 3】★★★ 46/60 — AI 复活+数字永生+墓碑二维码代理
- **信号源**：S3 跨世代刚需 + S7 AI 新刚需
- **本金**：1-3 万
- **预期**：5k-3万/月（客单价高）
- **打法**：联系宠物克隆厂商谈返点 + 1688 找 NFC 墓碑贴片 + HeyGen 数字人复活
- **反模式**：全 PASS
- **金句**："失去亲人的钱最好赚，因为那种痛是不计价的。"

### 【候选 4】★★★ 46/60 — Cross-Rollup MEV / 稳定币 DEX 套利
- **信号源**：S4 跨合规套利 + S1 跨语言价差

### 【候选 5】★★★★ 53/60 — Fiverr/Upwork 中文服务倒卖 + Temu 半托管

- **预期**：1-5k/月起步，3-6 个月达 1-3 万/月

- **反模式**：⚠ Temu 部分需要备货 5-10 万，先从 Fiverr 倒卖开始
